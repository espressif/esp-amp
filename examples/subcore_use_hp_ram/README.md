# ESP-AMP SubCore use HP RAM

This example demonstrates how ESP-AMP subcore project can use HP RAM for its firmware.

## How to use example

``` shell
source ${IDF_PATH}/export.sh
idf.py set-target <target>
idf.py build
idf.py flash monitor
```
