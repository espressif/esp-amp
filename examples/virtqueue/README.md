# ESP-AMP Virtqueue

This example demonstrates demonstrates how to send data from subcore (master core) to maincore (remote core) using virtqueue.

## How to use example

``` shell
source ${IDF_PATH}/export.sh
idf.py set-target <target>
idf.py build
idf.py flash monitor
```
